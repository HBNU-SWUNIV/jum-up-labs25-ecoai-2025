## Project Background
- 본 프로젝트는 자율주행 주차 로봇의 소프트웨어 성능을 테스트하고 최적화하는 것을 목표로 한다. 이를 위해 초기 단계에서 시스템 요구사항을 파악하고, 음성 명령 제어 시나리오를 도출했다. 또한, YOLO, CNN, ResNet 등 딥러닝 기반 컴퓨터 비전 기술에 대한 논문 조사를 통해 프로젝트에 적용 가능한 알고리즘의 장단점을 비교 분석했다.

## System Model
- 시스템은 로봇 운영체제(ROS)와 시뮬레이터(Gazebo)를 기반으로 구축되었으며, 크게 인식(Perception)과 제어(Control) 모듈로 구성된다.

시뮬레이션 환경은 Ubuntu 운영체제 상에 ROS1과 Gazebo를 구축하는 것으로 시작했다. TurtleBot3를 활용하여 노드(Node), 토픽(Topic) 등 ROS의 핵심 개념을 실습하였고, 가상 카메라 및 LIDAR 센서의 데이터 흐름을 파악했다.

인식(Perception) 모듈은 영상 인식과 음성 인식으로 나뉜다. 영상 인식은 YOLOv5를 초기 모델로 선정하여 주차 공간, 장애물 등의 객체 인식을 위한 데이터 수집 및 라벨링을 진행했으며, 이후 성능 향상을 위해 **YOLOv8-lite** 모델을 도입했다. 모델의 강인성을 확보하고자 야간, 역광, 우천 등 다양한 실제 환경 데이터를 수집하고 데이터 증강(Data Augmentation) 기법을 적용했다. 또한, 초기 시나리오를 바탕으로 한국어 음성 인식 및 명령 해석 모듈을 개발하여 고도화했다.

제어 및 경로 계획(Control & Planning) 모듈은 인식된 정보를 바탕으로 실제 로봇의 움직임을 생성한다. 허프 변환(Hough Transform) 파라미터 최적화와 이미지 필터링을 강화하여 다양한 조건에서 주차선을 검출하는 로직을 개선했다. 두 주차선 간의 간격을 측정하여 '주차 가능 공간'을 판단하는 로직을 구현했으며, Gazebo 환경에 평행, 수직 등 다양한 형태의 주차 공간을 추가하고 이를 자동 분류하는 기능을 개발했다. 인식된 주차 공간의 중앙을 최종 목표 지점으로 설정하고 로봇 기준 좌표계로 변환하는 알고리즘, 그리고 평행 및 수직 주차에 필요한 동작 순서(예: 전진-회전-후진)와 각 단계의 이동 거리/회전 각도를 계산하는 로직을 구현했다. 모든 제어 로직은 Gazebo 시뮬레이션 상에서 경로를 시각적으로 표시하여 사전 검증했다.


## Numerical Results
- 개발된 모델의 성능은 정량적 지표를 통해 체계적으로 분석하고 개선하는 과정을 거쳤다. YOLOv5 모델의 탐지 성능을 평가하기 위해 정밀도(Precision), 재현율(Recall), mAP(mean Average Precision), Confusion Matrix 등의 지표를 사용했다. 또한 실제 로봇 환경에서의 실시간성을 고려하여 추론 속도도 중요한 평가 지표로 관리했다. TensorBoard 시각화 분석을 통해 성능 저하의 원인을 파악했으며, 데이터 증강 기법을 적용한 재학습과 K-means 클러스터링을 활용한 Anchor Box 재산출 등을 통해 모델을 최적화했다.

## Conclusion
- 본 프로젝트를 통해 자율주행 주차 시스템의 성능이 데이터의 품질과 다양성에 결정적으로 의존함을 확인했다. 특히 야간, 우천 등 비정형 환경 데이터의 확보와 처리가 모델의 강인성을 확보하는 데 핵심적이었다. 또한, 시뮬레이션 환경에서의 성능과 실제 환경 간의 격차가 존재함을 발견했다. 시스템 통합 과정에서, 개별 모듈의 성능과 별개로 ROS 토픽을 통한 데이터 동기화 및 통신 지연 문제가 전체 시스템 안정성에 영향을 미칠 수 있음을 확인했다. 따라서 성공적인 자율주행 시스템은 정밀한 인식뿐만 아니라, 잘 정의된 목표 좌표계와 시뮬레이션을 통해 사전 검증된 실행 계획이 뒷받침되어야 한다는 결론을 도출했다.

## Project Outcome
- 프로젝트의 주요 성과는 다음과 같다.
1.  **YOLOv8-lite** 기반의 딥러닝 객체 검출 모델을 통해 주차선과 장애물 인식 정확도를 향상시켰다.
2.  전처리 및 학습 데이터셋 구성 과정을 자동화하여 다양한 조명, 기울기 환경에 대응하도록 개선했다.
3.  평행 및 수직 주차에 필요한 동작 시퀀스와 경로 계산 로직을 구현하고, 이를 ROS와 Gazebo 환경에 통합하여 시뮬레이션 검증을 완료했다.
