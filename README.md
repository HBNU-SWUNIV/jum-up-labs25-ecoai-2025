## Project Background
- 전 세계적으로 탄소 중립이 주요 과제로 대두됨에 따라, 대한민국 역시 2050년까지 탄소 중립 실현을 목표로 하고 있다. 이산화탄소 배출의 상당 부분은 화석 연료와 산업 활동에서 비롯되며, 이에 대한 해결책으로 산업체의 전력 데이터를 기반으로 최적의 솔루션을 제공하는 '그린버튼(Green Button)' 플랫폼이 주목받고 있다. 하지만 산업 공정 데이터는 다양한 환경 요인과 비정상적 운영으로 인해 신뢰성을 저해하는 이상치(anomaly)를 포함할 가능성이 높다. 본 연구는 이 산업체 전력 소비 데이터를 분석하고 이상치를 효율적으로 탐지하여 그린버튼 플랫폼의 신뢰성과 정확성을 높이는 방안을 제시하는 것을 목적으로 한다.

## System Model
- 본 연구는 산업체 전력 데이터의 이상치 탐지를 위해 데이터 전처리, 결측치 예측, 결과 분석의 3단계 프로세스 모델을 기반으로 한다.
먼저 데이터 전처리 단계에서는 15분 간격으로 수집된 산업체 전기 데이터를 확보한 뒤, 데이터 이상치 처리, 샘플링, 정규화 등 데이터 정제 및 변환 작업을 수행한다. 이후 데이터를 훈련(Training), 검증(Validation), 테스트(Testing) 세트로 분할한다.
다음으로 데이터 결측치 예측 단계에서는 전처리가 완료된 데이터를 기반으로 모델을 훈련한다. 특히 LSTM(Long Short-Term Memory) 모델을 활용하여 발생한 결측 데이터를 예측하는 과정을 수행한다.
마지막으로 예측 결과 분석 단계에서는 결측치 예측 모델이 도출한 예측 데이터와 실제 데이터를 비교 분석하여 모델의 성능을 검증한다. 이 과정과 더불어, 머신러닝 및 딥러닝 기반의 다양한 이상치 탐지 알고리즘을 개발하고 성능을 비교하며, 산업체별 특성에 가장 적합한 최적의 모델을 선정하는 비교 연구를 진행한다.

## Numerical Results
- 본 연구의 정량적 목표는 이상치 탐지 알고리즘의 성능을 극대화하여, 최종적으로 이상치 검출 정확도를 97%까지 향상하는 것이다. 이를 위해 다양한 알고리즘의 성능을 정량적으로 비교 평가하여 각 산업별 최적 모델을 도출한다.

## Conclusion
본 연구는 산업체 전력 데이터의 이상치 탐지 정확도를 향상시켜 탄소 중립 실현을 위한 에너지 관리 기술 발전에 기여하는 것을 목표로 한다. 신뢰도 높은 전력 데이터 분석은 산업체의 에너지 절감 및 온실가스 감축에 직접적으로 기여할 수 있다.

예를 들어, 식품 산업에서는 냉장 설비의 에너지 모니터링을 통해 불필요한 낭비를 줄이고 품질 관리를 최적화할 수 있으며, 금속 산업에서는 대규모 열에너지를 소모하는 공정의 피크 부하를 분산시켜 비용을 통제하고 설비 안정성을 확보할 수 있다. 화학 산업 역시 연속 공정의 에너지 투입량을 면밀히 계측하여 최적점을 설정하는 데 활용할 수 있다.

최종적으로 클라우드 기반의 플랫폼을 구축하여 대규모 데이터 처리, 실시간 이상 상황 대응, 기업별 맞춤형 모델 최적화를 지원하고, 이를 통해 비용 절감과 공정 안정화를 이룰 수 있다.

## Project Outcome
본 프로젝트의 최종 결과물은 산업 설비의 노후화 및 사고 위험을 사전에 예측하고 관측하여 공정 안정성을 높이는 이상치 탐지 플랫폼의 기반을 마련하는 것이다. 이를 통해 기업은 원가 절감과 탄소 배출 감소를 동시에 실현할 수 있다.
향후 이 연구는 신용카드 사기 거래 탐지, 네트워크 침입 탐지, 질병 조기 발견 등 다른 고차원 데이터의 이상 탐지 분야로 확장될 수 있는 기반 기술이 될 것이다.
